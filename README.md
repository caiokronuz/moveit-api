# Funcionalidades
## Rotas
/register (POST)
- Rota para registrar novos usuários

/login (POST)
- Rota para fazer o login de usuários

/status (PUT)
- Rota para atualizar o status do usuário

# Como utilizar

```console
git clone https://github.com/caiokronuz/moveit-api.git
```
```console
cd moveit-api/
```
```console
npm install
```
```console
npm run dev 
```
